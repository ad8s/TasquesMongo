## PROPOSTA
---

Proposta de base de dades per a una aplicació de tasques: la base de dades ha de permetre guardar tasques, on cada tasca té un identificador únic, un títol, un estat que indica si està pendent o finalitzada, i una data de creació; les tasques poden estar associades a un usuari mitjançant un identificador d’usuari, permetent que cada usuari gestioni les seves pròpies tasques de manera simple i directa.
